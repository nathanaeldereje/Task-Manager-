# Mobile Task Manager App built in Kotlin

## Features

- Add new tasks
- Mark tasks as completed or pending
- Delete tasks
- Filter tasks by:
  - All
  - Pending
  - Completed

## Tech Stack

- Kotlin (Android Studio)
- RecyclerView
- Toast for feedback
- MutableList for local state
